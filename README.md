# e-plantShopping

This repository contains the React application for the Paradise Nursery shopping cart project.  
The application allows users to browse a list of plants, add items to a cart, update quantities, remove items, and view the total cost.  
It uses React for the UI and Redux Toolkit for state management.

## Features
- Plant product listing
- Add to cart functionality
- Update item quantity
- Remove items from cart
- Cart total calculation
- Navigation between pages

## Technologies Used
- React
- Redux Toolkit
- Vite
- JavaScript
